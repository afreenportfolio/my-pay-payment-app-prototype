# MyPay: Payment App Prototype

![Type](https://img.shields.io/badge/Type-UI/UX_Internship-purple) 
![Status](https://img.shields.io/badge/Status-Completed-green)

MyPay is a UI/UX prototype for a mobile payment application, designed to streamline financial transactions. It prioritizes user-friendly interfaces with clean layouts, making payments secure and effortless.

## Features

1. Secure login and transaction flows.
2. Intuitive screens for bill payments and history.
3. Consistent iconography and design patterns.
4. Accessible and scalable components.

## Technologies used

- Design Tool: Figma
